# TheatreMaker
This is the repository for the paper: "TheatreMaker: Director-Actor Agent Collaboration for Controllable and Interactive Drama Script Generation".

Currently, we only provide the prompts used in our work. After organizing the work, we will release the full executable codes.

**April 2, 2024 Update:** We have updated some core codes of the TheatreMaker (architectures of director and actor agents). Currently, we are still in the process of organizing the work. The provided codes are not yet complete and lacks English comments, and do not represent the final open-source version.
